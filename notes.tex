Stacks:
\begin{itemize}
  \item Container of objects that are inserted and removed with last in first out (LIFO).
  \item Objects can be inserted any time, only most recent insert can be removed.
  \item Insert = push, remove = pop
\end{itemize}

Stack Operations:
\begin{itemize}
  \item new():ADT - creates new stack
  \item push(S:ADT,o:element):ADT - inserts object onto top of stack S
  \item pop(S:ADT):ADT - Removes the top object of stack S; if stack empty, error
  \item top(S:ADT):element - Returns the top object of stack, without removing; if empty, error
  \item size(S:ADT):integer - Returns number of objects in stack S
  \item isEmpty(S:ADT):boolean - Indicates if stack S is empty
\end{itemize}

EXAMPLE:
Algorithm computeSpans2(P):
Let D be an empty stack.
for i = 0 to n-1 do
h = 0; done = false;
while not (D is empty() or done) do{
if P[i] >= P[D. top()] then
D.pop()
else
done = true
}
if D is empty() then h = -1
else h = D.top()
S[i] = i-h
D.push(i)
return S

P = {1, 4, 2, 3, 8, 5, 6}
S = {1, 2, 1, 2, 5, 1, 2}
i = 0 to 6 (n-1)

i = 0:
h = -1, S[0] = 0-(-1) = 1

i = 1:
P[1] >= P[0]?
4       1
remove (pop) P[0] from stack, S[1] = 1 - (-1) = 2

i = 2:
P[2] >= P[1]?
2       4
S[2] = 2 - 1 = 1

i = 3:
P[3] >= P[1]?
3       4
S[3] = 3 - 1 = 2

i = 4:
P[4] >= P[1]?
8       4
S[4] = 4 - (-1) = 5
